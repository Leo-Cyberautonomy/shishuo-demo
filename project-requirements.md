# 知识付费平台（企业家×名师私域连接） - Demo 需求规格

## 项目概述
- **项目名称**：知识付费平台（暂定名：「师说」）
- **行业领域**：在线教育 / 知识付费 / 高端私域
- **项目简介**：一个连接企业家学员与顶级名师（北大、港大等知名高校教授）的知识付费平台。核心解决三个问题：①企业家有与名师 1 对 1 线上交流/上课的需求但缺乏渠道；②名师愿意授课赚取收入但碍于面子不便直接谈钱；③第三方培训组织需要高效对接师生资源。平台作为中间层，提供标准化的约课、支付、上课全流程服务，让知识交易体面且高效。

## 功能模块

### 模块 1：首页与发现
- **描述**：平台入口，展示精选名师、热门课程、平台调性，引导用户快速找到目标名师
- **页面**：
  - 首页 | 运营展示页 | 轮播 Banner、精选名师推荐、热门课程分类、平台数据概览（已服务企业家数、入驻名师数等）
  - 名师发现 | 筛选列表页 | 按学校/领域/评分筛选名师，卡片展示名师头像、头衔、擅长领域、评分、单价
  - 课程发现 | 筛选列表页 | 按分类（战略管理/金融投资/领导力/国学智慧等）浏览课程主题
- **关键交互**：
  - 搜索名师/课程关键词 → 实时搜索结果展示
  - 点击名师卡片 → 进入名师主页
  - 首页分类标签切换 → 动态加载对应名师列表

### 模块 2：名师主页与约课
- **描述**：名师的个人展示页面，展示名师资质、课程服务、可约时间，并完成预约下单
- **页面**：
  - 名师主页 | 详情页 | 名师照片、简介、学术背景、擅长领域、学员评价、可预约服务列表
  - 约课页 | 表单页 | 选择服务类型（1对1授课/咨询聊天/小班课）、选择时间段、填写需求描述、确认价格
  - 支付确认 | 表单页 | 订单摘要、支付方式选择、优惠券使用、确认支付
- **关键交互**：
  - 浏览名师主页 → 查看评价和服务列表 → 点击「立即预约」
  - 选择服务类型 → 日历选择可用时段 → 填写咨询主题 → 提交预约
  - 确认订单 → 选择支付方式 → 完成支付 → 显示预约成功

### 模块 3：在线课堂
- **描述**：1 对 1 或小班制线上授课场景，支持视频通话、屏幕共享、课件展示
- **页面**：
  - 课堂主界面 | 全屏功能页 | 视频画面（名师+学员）、课件/PPT 展示区、聊天互动区、课堂工具栏
  - 课前准备 | 信息页 | 课程主题、名师信息、设备检测（摄像头/麦克风）、课前资料查看
- **关键交互**：
  - 课前 15 分钟推送提醒 → 点击进入课前准备 → 设备检测通过 → 进入课堂
  - 课堂中：视频通话、屏幕共享切换、课件翻页、文字聊天、举手提问
  - 课程结束 → 弹出评价弹窗 → 提交评分和文字评价

### 模块 4：个人中心（学员端）
- **描述**：学员管理自己的预约、课程记录、收藏、消息和账户信息
- **页面**：
  - 我的课程 | 列表页 | 待上课、进行中、已完成三个 Tab，展示课程卡片（名师头像、课程主题、时间、状态）
  - 我的预约 | 列表页 | 预约记录，支持取消/改期操作
  - 学习记录 | 列表页 | 历史课程回放、课堂笔记、课后资料
  - 我的收藏 | 列表页 | 收藏的名师和课程
  - 消息中心 | 列表页 | 系统通知、课程提醒、名师回复
  - 账户设置 | 表单页 | 个人信息编辑、手机号绑定、支付管理、隐私设置
- **关键交互**：
  - 查看待上课列表 → 点击课程 → 进入课前准备/课堂
  - 查看已完课程 → 点击观看回放 → 下载课堂资料
  - 取消预约 → 确认取消 → 触发退款流程

### 模块 5：名师工作台（名师端）
- **描述**：名师管理自己的个人资料、服务设置、排课日历、收入和学员互动
- **页面**：
  - 工作台首页 | 仪表盘 | 今日课程安排、本月收入概览、待处理预约数、学员评价概览
  - 排课日历 | 日历页 | 可视化排课日历，设置可约时段、查看已约课程
  - 服务管理 | 表单列表页 | 创建/编辑服务（1对1授课、咨询、小班课）、设定价格、时长
  - 收入明细 | 列表页 | 收入统计图表、提现记录、结算明细
  - 学员管理 | 列表页 | 学员列表、上课次数、评价情况
  - 个人资料编辑 | 表单页 | 编辑个人简介、学术背景、擅长领域、头像照片
- **关键交互**：
  - 收到预约通知 → 确认/拒绝预约 → 系统通知学员
  - 在排课日历上设置空闲时段 → 学员端自动显示可约时间
  - 查看收入 → 申请提现 → 提现到账

### 模块 6：第三方机构入口
- **描述**：第三方培训组织可通过平台发布课程需求、批量组织学员约课
- **页面**：
  - 机构工作台 | 仪表盘 | 合作名师数量、已组织课程数、学员统计
  - 需求发布 | 表单页 | 发布培训需求（主题、人数、预算、时间范围），平台匹配推荐名师
  - 班级管理 | 列表页 | 管理学员分班、查看上课情况、导出学习报告
- **关键交互**：
  - 发布培训需求 → 平台推荐匹配名师 → 确认名师 → 批量创建课程
  - 查看班级列表 → 查看学员出勤和评价 → 导出报告

### 模块 7：平台管理后台 [推断]
- **描述**：平台运营方管理名师审核、订单管理、内容运营、数据统计
- **页面**：
  - 管理总览 | 仪表盘/大屏 | 平台核心指标（GMV、活跃用户、课程完成率、名师数量等）
  - 名师审核 | 列表页 | 名师入驻申请审核、资质验证
  - 订单管理 | 列表页 | 全平台订单查询、退款处理、争议处理
  - 内容运营 | 表单页 | Banner 管理、推荐名师配置、分类管理
  - 用户管理 | 列表页 | 学员/名师/机构账号管理
  - 财务管理 | 列表页 | 平台分成、提现审核、对账报表
- **关键交互**：
  - 审核名师申请 → 查看资质材料 → 通过/驳回 → 通知名师
  - 处理退款申请 → 查看订单详情 → 审批退款

## 可视化需求
- **图表类型**：
  - 折线图：收入趋势、用户增长趋势、课程完成率变化
  - 柱状图：各领域课程数量分布、月度订单量
  - 饼图：课程类型占比、学员行业分布
  - 仪表盘：名师评分展示
  - 日历热力图：名师排课日历、学员上课频次
- **大屏布局**：管理后台总览页采用数据仪表盘布局，核心指标卡片 + 趋势图 + 排行榜
- **3D/GIS**：无特殊 3D/GIS 需求

## UI 风格
- **主色调**：深蓝色（#1A365D）为主色，金色（#D4A853）为点缀色，传递高端、专业、信赖感
- **风格关键词**：高端商务 / 简约大气 / 私域精英 / 知性温润
- **参考说明**：
  - 目标用户是企业家群体，UI 需要传达高端感和专业度，避免花哨、廉价的视觉风格
  - 名师卡片设计参考高端社交名片风格，突出学术头衔和院校背景
  - 整体交互要流畅简洁，减少操作步骤，尊重企业家用户的时间
  - 参考产品：在行（专家约见）、得到（知识付费）、领英（商务社交）的高端调性
  - 移动端优先设计，企业家多在碎片时间使用手机操作

## Demo 交互要点
1. **名师发现与筛选全流程**：首页浏览 → 筛选名师 → 查看名师主页 → 阅读评价 → 点击预约，完整体现用户寻找名师的核心路径
2. **约课与支付闭环**：选择服务类型 → 选日期时间 → 填写咨询主题 → 确认支付 → 预约成功动效，体现平台解决"体面付费"问题的核心价值
3. **在线课堂体验**：课前准备（设备检测）→ 进入课堂（视频+课件双画面）→ 课后评价，展示线上授课的完整体验
4. **名师工作台排课**：日历视图设置可约时段 → 收到预约通知 → 确认预约 → 查看收入统计，展示名师端的操作体验
5. **平台数据大盘**：管理后台的数据总览仪表盘，展示平台运营能力（名师数、学员数、GMV、课程完成率等核心指标的动态展示）
6. **机构批量约课**：机构发布培训需求 → 平台推荐名师 → 确认后批量创建课程，展示 B 端合作场景
